## 📘 Day 17: Nested Loops, Break, Continue, Labels

### 🔹 Nested For Loops
- A loop inside another loop.  
- Syntax:
 
  for(initialisation; condition; updation) {
      for(initialisation; condition; updation) {
          // statements
      }
  }

- Example:

  for(int i = 1; i <= 3; i++) {
      for(int j = 1; j <= 2; j++) {
          System.out.println("i=" + i + ", j=" + j);
      }
  }

- ⚠️ Not used much in real-time unless working with matrices, patterns, or nested data.
