### Day 1: Introduction to Programming

- **What is Programming?**  
  Programming is the process of giving a computer a set of instructions to perform specific tasks.

- **Types of Programming Languages:**
  - **High-Level Languages:** These are human-readable, easy to understand, and include examples like Java, Python, and C++.
  - **Low-Level Languages:** These are closer to machine understandable language,  and not as easy for humans to read.

- **Translators:** These convert high-level code into machine code. There are two main types: compilers and interpreters.
