
Day 26 – Java Arrays (Summary)

Arrays

An array is used to store multiple values of the same data type in a single variable.

Instead of creating many variables, we can store all values in one array.


---

Array Declaration & Creation

When creating an array, you must specify its size.

Syntax:

int[] marks = new int[5];

int → Data type of elements.

marks → Array name.

5 → Size (number of elements).



---

Memory Allocation

Memory allocated for an array depends on:

> Size of one data type × Number of elements (array length)



Example:

int[] marks = new int[5];

If an int occupies 4 bytes:

4 × 5 = 20 bytes

(Conceptually. The JVM also has some object overhead, but for learning, this formula is sufficient.)


---

Indexing

Array indices start from 0.


Example:

marks[0]
marks[1]
marks[2]
marks[3]
marks[4]

For an array of size 5:

First index = 0

Last index = length - 1 = 4



---

Default Values

If values are not assigned, Java automatically initializes the array with default values.

Primitive Types

Data Type	Default Value

byte	0
short	0
int	0
long	0L
float	0.0f
double	0.0
char	'\u0000' (null character)
boolean	false


Non-Primitive Types

Default value is null.


Example:

String[] names = new String[3];

Output:

null
null
null


---

Exceptions

1. ArrayIndexOutOfBoundsException

Thrown when you access an index that is outside the valid range.

Example:

int[] marks = new int[5];

System.out.println(marks[5]); // Exception

Valid indices are only:

0 to length - 1


---

2. Negative Index

Negative indices are not allowed.

Example:

marks[-1];

This also throws ArrayIndexOutOfBoundsException.


---

Quick Revision

✅ Arrays store same type of data.

✅ Size must be specified during creation.

✅ Syntax:

int[] marks = new int[5];

✅ Index starts from 0.

✅ Last index = length - 1.

✅ Default values are automatically assigned.

✅ Primitive defaults:

int → 0

float → 0.0f

double → 0.0

boolean → false

char → '\u0000'


✅ Non-primitive default → null.

❌ Accessing an index < 0 or >= length throws ArrayIndexOutOfBoundsException.
