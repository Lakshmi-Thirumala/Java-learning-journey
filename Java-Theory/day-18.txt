## 📘 Day 18:  Break, Continue, Labels(Jump statements)

### 🔹 Break in Loops
- Exits the **current loop** immediately when condition is met.  
- Example:

  for(int i = 1; i <= 3; i++) {
      for(int j = 1; j <= 3; j++) {
          if(j == 2) break; // exits inner loop
          System.out.println("i=" + i + ", j=" + j);
      }
  }

- ✅ Comes out of the **inner loop** only.

---

### 🔹 Continue in Loops
- Skips the current iteration and moves to the next.  
- Example:
 
  for(int i = 1; i <= 3; i++) {
      for(int j = 1; j <= 3; j++) {
          if(j == 2) continue; // skips j=2
          System.out.println("i=" + i + ", j=" + j);
      }
  }


---

### 🔹 Labels in Loops
- You can give a **name** to a loop and use it with `break` or `continue`.  
- Syntax:
 
  labelName:
  for(initialisation; condition; updation) {
      for(initialisation; condition; updation) {
          if(condition) break labelName;
          // statements
      }
  }
 
- Example:
 
  outer:
  for(int i = 1; i <= 3; i++) {
      for(int j = 1; j <= 3; j++) {
          if(j == 2) break outer; // exits outer loop
          System.out.println("i=" + i + ", j=" + j);
      }
  }

---

### 🔹 Control Flow Statements
These are special statements that alter the normal flow of execution in loops or conditional blocks.
break and continue are control flow statements that can be used in any loop structure (for, while, do-while) and also inside a switch statement.

1. **break statement**  
   - Official name: **Break Statement**  
   - Purpose: Terminates the loop or switch immediately and transfers control to the statement after the loop/switch.  

2. **continue statement**  
   - Official name: **Continue Statement**  
   - Purpose: Skips the current iteration of the loop and jumps to the next iteration.  

3. **return statement** (related but broader)  
   - Official name: **Return Statement**  
   - Purpose: Exits from a method and optionally returns a value.  

4. **labels**  
   - Official name: **Labeled Statements**  
   - Purpose: Give a name to a loop, so `break` or `continue` can target that specific loop (useful in nested loops).  

---

### ✅ Day 17 Summary
- **Nested loops**: loop inside another loop, useful for patterns/matrices.  
- **Break**: exits the current loop.  
- **Continue**: skips current iteration.  
- **Labels**: allow breaking/continuing outer loops directly.
